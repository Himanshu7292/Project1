#Console Based Shopping App
 JoyBox App
#Technologies Used
* Java
* Maven
* JBDC
*J Unit
* MySql
* Log4J
#Devloped console based application with some functionalities,
* ##Customer Functionalities
   * Register
   * Login
   * View products
   * Add products to  cart
   * View cart
   * Logout
* ##Employee Functionalities
   * Login
   * Add new products
   * View Customers
   * Update products price
   * Logout
